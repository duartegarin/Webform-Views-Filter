INTRODUCTION
------------

Current Maintainer: Duarte Garin <duartegarin@fastmail.com.au>

By using this module you can filter a view to show only content that has webform
components attached to it,
or by using the negative operator only content that does not have components
attached.


INSTALLATION
------------

This module is installed as any other drupal module.
See http://drupal.org/node/70151 for instructions.
After the module is enabled you will see a new type of filter in your views
under the Webform category.
